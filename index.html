UFO Data Visualization

UFO Sightings Over Time

This visualization shows the number of UFO sightings over time. The x-axis represents the year, derived from the datetime variable, and the y-axis shows the total number of sightings. A line chart was chosen to highlight trends over time.

UFO Sightings by Shape

This visualization shows the number of UFO sightings by shape using a bar chart. The x-axis represents UFO shape categories, and the y-axis shows the count of sightings. Color is used to distinguish between categories.

Interactivity was added through a dropdown menu that allows users to filter sightings by shape. This helps users explore the data more deeply and focus on specific categories.
