UFO Data Visualization

UFO Sightings Over Time

This visualization shows the number of UFO sightings over time. The x-axis represents the year, which was extracted from the datetime variable using pandas, and the y-axis shows the total number of sightings. A line chart was chosen because it effectively displays trends and patterns over time, making it easier to observe increases or decreases in sightings. Data transformation included converting the datetime column to a proper datetime format and extracting the year for aggregation.

UFO Sightings by Shape

This visualization shows the number of UFO sightings by shape using a bar chart. The x-axis represents UFO shape categories, and the y-axis shows the count of sightings. A bar chart was chosen because it allows for clear comparison between categorical values. Color encoding is used to distinguish between different shape categories, improving readability.

Interactivity was added through a dropdown menu that allows users to filter sightings by shape. This feature enables users to explore specific categories more closely and reduces visual clutter when focusing on particular shapes.
